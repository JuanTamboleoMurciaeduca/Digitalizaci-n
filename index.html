Redes LAN (Local Area Network)

Una red de área local (LAN) es un conjunto de dispositivos interconectados en un área limitada, como una casa, oficina o escuela.

Tipos de Redes LAN

  • LAN cableada
  • LAN inalámbrica (WLAN)

Topologías de Redes LAN

  • Estrella
  • Bus
  • Anillo
  • Malla

Dispositivos en una LAN

  • Switch
  • Router
  • Access Point
  • Computadoras y servidores

Ventajas de las Redes LAN

  • Alta velocidad de transferencia de datos
  • Compartición de recursos
  • Seguridad y control centralizado

Ejemplos de Uso

  1. Redes en oficinas
  2. Redes en universidades
  3. Redes en hogares

Términos Clave

Término Definición
IP Dirección única para identificar dispositivos en la red.
MAC Identificador físico de un dispositivo.
DHCP Protocolo para asignar direcciones IP automáticamente.
